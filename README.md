# 🦸‍♀️ Number Hero Academy

Number Hero Academy is an advanced, fully offline Progressive Web App (PWA) built specifically for children (ages 5-10) to master numbers from 1 to 10,000. 

Unlike standard rote-learning apps, it uses smart AI-like heuristics and scientifically proven educational concepts like Spaced Repetition (SRS) to ensure genuine number sense.

### ✨ Key Features
* 📱 **100% Offline PWA:** Installable on Android/iOS and works without the internet.
* 📈 **3,000 Adaptive Levels:** Smooth scaling difficulty from single digits to 10,000.
* ✍️ **Smart Handwriting Engine:** Evaluates tracing accuracy offline without heavy AI models.
* 🧩 **Multiple Game Modes:** Missing numbers, Bada/Chhota (Compare), Even/Odd, Base-10 blocks, and Time Attack!
* 📊 **Deep Parent Analytics:** Tracks exact mistakes, weak areas, and generates printable PDF progress reports.
* 🧠 **Dyslexia-Friendly & Accessible:** Toggleable OpenDyslexic font and voice guidance.

Built purely with HTML, CSS (Tailwind), and vanilla JavaScript. No backend required!
